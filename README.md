# visualdon-projet
### Contexte : 
*d'où viennent les données, qui les a créées et dans quel contexte ? Qui a décidé de les collecter, et pourquoi ? Quels biais ou absences peut-on identifier ?*

### Description : 
*Comment sont structurées les données ? Parler du format, des attributs et du type de données.*

### But : 
*souhaitez-vous explorer (rechercher des tendances, poser des questions ouvertes) et/ou expliquer (raconter quelque chose de précis, guider le lecteur) ? Les deux approches peuvent coexister. Quelle histoire voulez-vous raconter ? Quel regard portez-vous sur ces données ?*

En tant que joueurs nostalgiques de Pokémon, nous souhaitons retracer l'évolution de la licence afin de mettre en lumière les dérives progressives qui ont suscité frustration et déception au sein de la communauté historique des fans.

### Références : 
*Qui d'autre dans le web ou dans la recherche a utilisé ces données ou abordé ce sujet ? Dans quel but ? Cherchez des exemples sur des plateformes comme The Pudding, dans la presse de données, ou dans des projets académiques.*
